# EmailParser

## Le sujet:
Récupération des courriels à partir d’une boîte mail

Le script est écrit en Python

## Ce script permet de :
-récupérer les nouveaux mails non lus et les marque en tant que mail lu
-Lire les nouveaux mails et les sauvegarde en fichier json dans un répertoire paramétrable

## Le fichier conf.txt contient:
-l'adresse mail
-le mot de passe
-le nom du serveur Imap
-l'adresse du dossier de sauvegarde des mails

Pour trouver votre addresse imap :
https://knowledge.hubspot.com/fr/email-notifications/how-can-i-find-my-email-servers-imap-and-smtp-information
